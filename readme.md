# SDL2 maps experiment

## Build/Run
```bash
# build
make

# run
make run
```

---

### Linux
```bash
# dependencies
apt install -y libsdl2-dev libsdl2-image-dev
```
### Windows
- setup mingw like https://github.com/skeeto/w64devkit

- install dependencies
```bash
# inside of a shell (bash)
./download_nt.sh --curl --sdl2 --sdl2-image
```
